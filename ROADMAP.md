# Roadmap

Cloudstone is developed in dependency-ordered phases against the normative
[spec](../spec), with conformance measured by the TCK. Planning happens
privately; a phase appears here when work on it starts, and design
documentation for implemented behaviour is published as each phase ships.

| Phase | Scope | Status |
|---|---|---|
| 0 | Groundwork: module skeleton, fail-closed tenant authentication, H2/Postgres schema, zero-dependency local run | in progress |

Further phases are planned and will be listed as they begin. No dates.
